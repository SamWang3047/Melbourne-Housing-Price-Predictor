How to use the forecast page

The main page lets you search for a Melbourne locality and instantly view a quarterly price trend with the next-quarter forecast.

  1. Open the main page at /.
  2. Type a locality name such as Camberwell.
  3. Choose one of the autocomplete suggestions.
  4. Click Generate Forecast or press Enter.
  5. Read the summary cards above the chart.
  6. Use the chart to compare the historical blue series with the red forecast segment.

The chart shows:

  1. Blue line: historical quarterly prices
  2. Red dashed line: forecast segment
  3. Price label on every node
